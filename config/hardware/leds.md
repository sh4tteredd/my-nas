# LED Configuration

This document provides a brief overview of the LED configuration used on my UGREEN DXP2800.

## Controller Tool

- **Project:** [UGREEN LEDs Controller](https://github.com/0x556c79/install_ugreen_leds_controller)
- **Purpose:** Restoring and managing front panel LED functionality

## Implementation

To get the front panel LEDs working correctly, I used the **install_ugreen_leds_controller** script from this repo and just followed all the instructions. 

When you start tweaking the system or move away from the stock OS, the front LEDs usually stop working or fail to reflect the actual hardware status. I applied this script **early on** to get the power, network, and disk activity lights behaving exactly the way they should. 

The setup was straightforward and it brings the hardware back to life so you can actually read the NAS status at a glance. Just like the RAM upgrade, I think it's an obligated update if you are customizing the software on this machine.
